# icon-xprt

It's a clone of [icon-sets.iconify.design](https://icon-sets.iconify.design/).

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm install
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
